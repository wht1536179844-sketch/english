## 二、每日自动更新机制（不依赖 Claude）

链路：Windows 计划任务（每天 7:30，错过开机补跑）
→ 运行 `每日更新.bat`
→ 调用本机 Codex CLI（ChatGPT 会员）
→ 按 `每日播报_操作规程.md` 抓新闻/热榜/播客
→ 更新 `早报数据.js`
→ git push → 网页自动更新。

前提条件（缺一不可）：
1. 电脑开机、有网（当天 7:30 没开机，下次开机自动补跑）；
2. Codex 已登录（`codex login` 登录一次长期有效）；
3. GitHub 凭据有效（已存在 Windows 凭据管理器，一般不会失效）。

## 四、常见故障排查

1. **网页没更新** → 看本文件夹 `日报运行日志.txt` 最后一段：
   - 日志里没有当天记录 → 计划任务没跑：任务计划程序里找「每日世界早报更新」，右键"运行"测试；
   - 有记录但报错 codex 未登录 → 终端运行 `codex login` 重新登录；
   - 有记录但 push 失败 → 终端在本文件夹运行 `git pull --rebase` 后 `git push`。
2. **句子诊所查不了语法** → LanguageTool 免费接口偶尔限流，稍后重试即可；一直不行多半是网络问题。
3. **想手动补跑一次早报** → 双击 `每日更新.bat` 即可。
4. **想改定时时间** → 任务计划程序 →「每日世界早报更新」→ 触发器改时间。

## 五、以后想改内容怎么办（用任何 AI 都行）

- 语法地图加内容：让 AI 编辑 `英语语法大地图.html` 里的 `grammarMap` 数据（有注释说明格式），改完 `git add + commit + push`；
- 换人物志播客单/加故事：编辑 `人物志故事库.js` / `人物志播客单.js`，同上推送；
- 调整早报板块：改 `每日播报_操作规程.md` 即可，第二天生效，不用改代码。
